# termenal-web specifications

This folder is the authoritative specification store for `termenal-web`.

Documentation convention (applies to this repository):

1. **`AUTHOR-LINGUISTIC-SPECIFICATIONS.md`** records each author (owner)
   specification **verbatim** — the exact request that defines a standard or
   requirement, with its original wording preserved.
2. Each author specification is translated into **executable, code-linked
   directives** in **`directives/SPEC-NNNN-AI-DIRECTIVES.md`**. The directives
   document links every directive to the concrete code / execution files that
   implement or verify it.

| Document | Purpose |
| --- | --- |
| [`AUTHOR-LINGUISTIC-SPECIFICATIONS.md`](AUTHOR-LINGUISTIC-SPECIFICATIONS.md) | Verbatim author specifications (SPEC-0001…) |
| [`directives/SPEC-0001-AI-DIRECTIVES.md`](directives/SPEC-0001-AI-DIRECTIVES.md) | Executable, code-linked directives that implement the specifications |
