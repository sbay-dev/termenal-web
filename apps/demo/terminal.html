الصفحة تحمّل الخطوط الافتراضية وتتّصل بالصدفة تلقائيًّا (شرط تشغيل الجسر المحلي npm run server على ws://127.0.0.1:5179). انقر داخل الطرفية واكتب أوامرك — مثل dir أو Get-Date أو echo مرحبا. الأسطر العربية تُحاذى تلقائيًّا إلى اليمين. مرّر بعجلة الفأرة لتصفّح السجل، اسحب لتحديد النص. الزر الأيمن ينسخ التحديد (ثم يزيله) أو يلصق عند المؤشر إن لم يكن هناك تحديد، مع إشعار يؤكّد العملية. Ctrl+C يقاطع الأمر الجاري (SIGINT) أو ينسخ التحديد إن وُجد، وCtrl+V يلصق. زر ملء الشاشة يحافظ على أبعاد ثابتة بنسبة ذهبية مستقرة عند التكبير. الخطوط مضمّنة برخصة OFL وتُقرأ محليًّا.
